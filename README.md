# Skill-Matrix

A web application designed to compile and map the skills of employees within a team or organization. This tool provides a clear and detailed list of employee skills and their proficiency levels, enabling managers to get an overall view of the capabilities available on their team.

## Features

* **Skill Mapping:** A centralized database to document and track employee skills.
* **Proficiency Levels:** Detailed proficiency ratings for each skill to provide a nuanced understanding of employee capabilities.
* **Team Overview:** A comprehensive dashboard for managers to view and analyze team skills.
* **User Profiles:** Individual user profiles with contact information, role, certifications, and a personal summary.
* **Team Management:** Functionality to create and manage teams, and assign employees to them.
* **Search and Sort:** Easily search for employees and sort skills based on proficiency.
* **Categorized Skills:** Skills are organized into categories such as "Maths, science and engineering," "Soft skills (& civic competence)," "Digital competence," and "Entrepreneurship (Business)."

## Technologies Used


- **Backend**: .NET (C#), ASP.NET Core  
- **Database**: (e.g. SQL Server / SQLite / PostgreSQL)  
- **Frontend**: Razor / MVC 
- **ORM / Data Access**: Entity Framework or similar  
- **Other**: Dependency Injection, DTOs, Middleware, etc.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* .NET SDK
* A code editor like Visual Studio or VS Code

### Installation

1.  **Clone the repo**
    ```sh
    git clone [https://github.com/Oriolowo-Mustapha/Skill-Matrix.git](https://github.com/Oriolowo-Mustapha/Skill-Matrix.git)
    ```
2.  **Navigate to the project directory**
    ```sh
    cd Skill-Matrix
    ```
3.  **Restore dependencies**
    ```sh
    dotnet restore
    ```
4.  **Run the application**
    ```sh
    dotnet run
    ```

## Usage

Once the application is running, open your web browser and navigate to the provided local URL. You can then:

* Create and manage user profiles.
* Add and categorize skills.
* Create and manage teams.
* View the skills matrix to see an overview of team capabilities.
